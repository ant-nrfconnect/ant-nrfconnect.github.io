CONFIG_ANT_BSC_LOG_LEVEL

(No prompt – not directly user assignable.)

Type: int

Direct dependencies

LOG && ANT_BSC && ANT

(Includes any dependencies from ifs and menus.)

Defaults

  • 0 if ANT_BSC_LOG_LEVEL_OFF

  • 1 if ANT_BSC_LOG_LEVEL_ERR

  • 2 if ANT_BSC_LOG_LEVEL_WRN

  • 3 if ANT_BSC_LOG_LEVEL_INF

  • 4 if ANT_BSC_LOG_LEVEL_DBG

  • LOG_DEFAULT_LEVEL if ANT_BSC_LOG_LEVEL_DEFAULT
