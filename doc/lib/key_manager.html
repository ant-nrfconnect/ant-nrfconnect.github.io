ant_err_t ant_plus_key_set(uint8_t network_number)

Function for registering an ANT+ network key.

The ANT_PLUS_NETWORK_KEY has been predefined in this module for exclusive use by ANT+ Adopters. It is to be used by devices that will communicate on the ANT+ network and strictly adhere to the ANT+ Device Profiles.

Parameters

  • network_number – [in] Network key number.

Returns

An ANT library error code.

ant_err_t ant_fs_key_set(uint8_t network_number)

Function for registering an ANT-FS network key.

The ANT_FS_NETWORK_KEY has been predefined in this module for exclusive use by ANT+ Adopters. It is to be used strictly for ANT-FS applications.

  • network_number – [in] Network key number.
