Welcome to ANT for nRF Connect SDK!

This page contains the resources you need to create ANT and ANT+ enabled applications with the nRF Connect SDK.

ANT for nRF Connect SDK is available on GitHub from ANT Wireless, a division of Garmin Canada Inc.: https://github.com/ant-nrfconnect/sdk-ant/

The nRF Connect SDK is available on GitHub from Nordic Semiconductor: https://github.com/nrfconnect/sdk-nrf
